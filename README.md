# Manual Testing of ChatGPT Web Application

## Project Overview

This repository demonstrates a comprehensive **Manual Software Testing** project for the **ChatGPT Web Application**. The testing was conducted to verify the application's functionality, usability, responsiveness, accessibility and overall user experience.
The project follows the complete Software Testing Life Cycle (STLC), including test planning, test scenario creation, test case design, test execution, bug reporting, exploratory testing and test summary reporting.

---

## Project Objectives

- Validate the core functionalities of the ChatGPT Web Application.
- Identify functional and non-functional defects.
- Verify that the application provides a smooth and user-friendly experience.
- Practice industry-standard manual testing documentation.
- Demonstrate practical knowledge of the Software Testing Life Cycle.

---

## Application Under Test

**Application Name:** ChatGPT

**Application Type:** Conversational AI Assistant

**Testing Type:** Manual Testing

**Test Environment:** Windows 11, Google Chrome

**Internet Connection:** Broadband & Slow Network Simulation

**Website:** https://chatgpt.com/
